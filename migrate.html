TaskFlow — Data Migration

Import your existing Google Sheets data into Supabase. One-time use only.

1. Supabase Connection

Enter your Supabase project URL and service_role key (NOT anon key — service_role bypasses RLS for migration).

2. Upload CSV Files

In each Google Sheet tab, go to File → Download → CSV. Then select each file below.

3. Run Migration

This will insert all data into your Supabase tables. Existing data won't be affected (inserts only, no deletes).
